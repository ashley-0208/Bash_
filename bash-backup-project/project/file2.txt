| بخش          | توضیح                                                          |
| ------------ | -------------------------------------------------------------- |
| `tee -a`     | باعث میشه خروجی همزمان در ترمینال و فایل لاگ نوشته بشه.        |
| `log`        |  تابع   | همه‌ پیام‌ها رو با تاریخ و ساعت مدیریت می‌کنه.                 |
| `backup.log` | توی همون دایرکتوری پروژه ساخته میشه. می‌تونی مسیرشو تغییر بدی. |
